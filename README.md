# Hairqoo Webpage

Landing page projektu Hairqoo (PL/EN) w wersji statycznej: `HTML + CSS + JS`.

## Uruchomienie lokalne

### Opcja 1: Python (najprościej)

```bash
python -m http.server 5500
```

Następnie otwórz:

`http://localhost:5500`

### Opcja 2: VS Code / Cursor Live Server

1. Otwórz folder repozytorium.
2. Uruchom `index.html` przez Live Server.
3. Wejdź na adres podany przez rozszerzenie.

## Struktura plików

- `index.html` - markup strony i sekcje landing page
- `styles.css` - styl wizualny i responsywność
- `script.js` - animacje, przełącznik języka PL/EN, logika formularza
- `*.png` - assety graficzne używane na stronie

## Deploy

Projekt jest gotowy do hostowania jako statyczna strona (np. GitHub Pages).
